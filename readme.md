# AI Resume Analyzer

An AI-powered Resume Analyzer built using Flask, Ollama, Sentence Transformers, and semantic similarity matching.

This application allows users to upload resumes in PDF format and analyze them using local AI models.

---

# Features

- Resume PDF Upload
- PDF Text Extraction
- Email Extraction
- Phone Number Extraction
- Skill Detection
- AI Resume Analysis
- ATS-style Resume Scoring
- Job Description Matching
- Semantic Similarity Matching
- Local AI using Ollama + Mistral

---

# Technologies Used

- Python
- Flask
- Ollama
- Mistral AI
- Sentence Transformers
- Scikit-learn
- PyPDF

---

# How It Works

1. Upload Resume PDF
2. Enter Job Description
3. AI extracts resume information
4. Skills are detected automatically
5. Resume score is calculated
6. Semantic similarity is computed
7. AI generates analysis and recommendations
